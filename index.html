Lecture 3.3 Controllers

- WebXR Input Source (a controller) as 2 transformation matrices that allow us to manage 6 degrees of freedom (position + orientation) + event listeners
- One transformation matrix defines the target ray space (the guardian boundary).
- The other defines the grip space. This defines the position of the model that visualizes the actual controller.
